# Security Policy

## Supported Versions

This project is actively maintained on the latest version of the `main` branch.

| Version | Supported |
|--------|-----------|
| Latest | ✅ |
| Older versions | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

📧 Contact: **yahyaegz@gmail.com**

Please include:

- A clear description of the vulnerability
- Steps to reproduce the issue
- Any potential impact
- Suggested fixes (if available)

I will review reports as soon as possible and provide updates on the status of the issue.

Thank you for helping keep this project secure.
