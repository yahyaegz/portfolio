# 🚀 About The Project

A **modern developer portfolio** built with a **Vite + Tailwind frontend** and a **.NET Clean Architecture backend**.

It showcases projects, skills, and provides a contact system for recruiters and collaborators.

---

# 🧠 Architecture

```id="arch1"
portfolio
│
├── frontend
│   ├── components
│   ├── pages
│   ├── sections
│   └── styles
│
└── backend
    ├── Domain
    ├── Application
    ├── Infrastructure
    └── API
```

# 🚀 Installation

### Clone Repository

```bash id="clone1"
git clone https://github.com/YOURUSERNAME/portfolio.git
cd portfolio
```

### Install Frontend

```bash id="front1"
npm install
npm run dev
```

### Run Backend

```bash id="back1"
dotnet run
```
